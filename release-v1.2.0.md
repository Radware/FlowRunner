# FlowRunner v1.2.0

FlowRunner v1.2.0 improves execution control, adds navigation tools for large flows, and makes results easier to export and share.

## Key Additions
- Auto Arrange layout for the Node-Graph view.
- Step search with jump to list or graph.
- Export execution results to JSON or CSV.
- Copy cURL resolves static variables, runtime values (when available), and special random variables.
- New special variables: `RANDOM_INT` and `RANDOM_STRING`.

## Fixes
- Stop now halts step-by-step runs and clears queued steps.
- Fixed the Previous/Next label for the last node in the step editor.
- Insert Variable button text no longer overflows.

## Installation

Download the installer matching your OS:

- **Windows (x64):** `FlowRunnerSetup-x64-win-1.2.0.zip`
- **macOS (arm64):** `FlowRunnerSetup-arm64-mac-1.2.0.dmg`

Follow the same installation steps as previous versions.

## Compatibility

Flows created in earlier versions remain fully compatible with v1.2.0.
